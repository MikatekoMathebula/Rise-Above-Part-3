Rise Above Nonprofit Website

Project Overview

Rise Above is a nonprofit organization website developed using HTML. The website is designed to raise awareness about drug addiction and provide information about recovery support services. It also encourages community involvement through volunteering, donations, and awareness campaigns.

Purpose of the Website

The purpose of this website is to:

* Educate users about drug addiction and recovery
* Provide information about available support services
* Promote community awareness and outreach programs
* Offer a way for users to contact the organization

Target Audience

This website is intended for:

* Individuals struggling with addiction
* Families affected by substance abuse
* Community members seeking information
* Volunteers and donors
* Organizations interested in partnerships

Website Structure

The website consists of five main pages:

* Home (index.html): Introduction, mission, vision, and impact
* About (about.html): Background, values, team, and goals
* Services (services.html): Details of support services offered
* News (news.html): Updates, events, and activities
* Contact (contact.html): Contact information and enquiry form

Technologies Used

* HTML5
* Basic web structure elements (headings, paragraphs, lists, forms, images)

Features

* Multi-page website structure
* Navigation bar for easy page access
* Informative content about the organization
* Contact form for user interaction
* Use of images to enhance visual appeal

Folder Structure

/images → Contains all images (logo, outreach, counselling, etc.)
index.html → Home page
about.html → About page
services.html → Services page
news.html → News page
contact.html → Contact page

How to Run the Project

1. Download or copy all project files
2. Open the folder on your computer
3. Double-click on **index.html**
4. The website will open in your browser
